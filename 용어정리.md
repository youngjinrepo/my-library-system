# 용어정리

| 용어              | 설명                         |
|-----------------|----------------------------|
| 서지 정보(BookInfo) | 책의 제목, 저자, ISBN 등 책의 공통 정보 |
| 소장 도서(BookCopy) | 서가에 꽂힌 실제 책                |
| 이용자(Memeber)    | 도서관의 서비스를 이용하는 사람          |
| 대출(Loan)        | 책을 빌려가는 행위                 |
| 반납(Return)      | 책을 반납하는 행위                 |
